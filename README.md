

| Команда | Описание |
| :--- | :--- |
| `/create <name>` | Создать новую группу и автоматически войти в нее |
| `/join <name>` | Войти в существующую группу по имени |
| `/leave` | Покинуть текущую активную группу |
| `/delete <name>` | Удалить указанную группу |
| `/list_groups` | Показать список всех активных групп |
| `/send_group <grp> <msg>` | Отправить сообщение в конкретную группу (не входя в нее) |
| `/send_user <user> <msg>` | Отправить личное сообщение конкретному пользователю |
| `<текст>` | Отправить сообщение в текущую активную группу |
